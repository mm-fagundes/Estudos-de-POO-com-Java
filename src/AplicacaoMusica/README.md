<h3>Projeto de finalização do curso Java: aplicando a orientação a objetos.</h3>
<h2>Mini aplicação de serviço de streaming de áudio fictício.</h3>
<h3>Nessa aplicação, o usuário pode, reproduzir, curtir, e ver dados de um áudio, como os cantores, produtoras e etc.
<br> O usuário pode também, ver o quão relevante um áudio é, de acordo com as regras de negócio pré-estabelecidas.</h2>



<h4>Concluído ✓</h4> 
Objetivo: 

- Criar uma aplicação de áudios ✓

- Possuir uma SuperClasse chamada Audio ✓
- Possuir subclasses podendo ser Musicas, Podcasts, etc. ✓

Classe Audio:
- Título ✓
- Duração ✓
- Total de reproduções ✓
- Curtidas ✓
- Classificação ✓

Metodos:
- Curtir ✓
- Reproduzir ✓
- Exibir dados(Autor, Descrição) ✓
